HTML

  • HTML is used to create the structures of a webpage, and it is made up of elements.
  • An element is a type of content on a webpage.
  • HTML attributes define additional information an element.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child element.
  • The elements that holds all elements are known as parent element
  • It is advisable to use semantic HTML elements as they make HTML easier to read by developers and browser.
  • However, all elements are semantic.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: adds changes to the staging area of Git
  • git commit -m "short description": commits changes in a descriptive way
  • git pull origin main: pull the source code from the GitHub to ensure we have the latest commit changes
  • git push origin nameoffeaturebranch: pushes changes in the feature branch to GitHub
