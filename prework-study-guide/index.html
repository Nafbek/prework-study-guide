HTML

  • HTML is used to create the structures of a webpage, and it is made up of elements.
  • An element is a type of content on a webpage.
  • HTML attributes define additional information an element.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child element.
  • The elements that holds all elements are known as parent element
  • It is advisable to use semantic HTML elements as they make HTML easier to read by developers and browser.
  • However, all elements are semantic.

CSS

  • Cascading Style Sheet (CSS) defines how the webpage content is presented to users.
  • There are three common ways to style a webpage using CSS.
  • These are:
    • Inline CSS
    • Internal CSS
    • External CSS
  • External CSS style sheet is the most common as it allows developers to keep all their CSS rules in a separate file.
  • We have to link the file using a element placed in the HTML's if we use an external style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border property goes around the padding and content.
  • Declaration contains two important components:
    • CSS Property, and
    • Value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: adds changes to the staging area of Git
  • git commit -m "short description": commits changes in a descriptive way
  • git pull origin main: pull the source code from the GitHub to ensure we have the latest commit changes
  • git push origin nameoffeaturebranch: pushes changes in the feature branch to GitHub

Javascript

  • JavaScript is a dynamically typed language.
  • A variable is a named container that allows us to store data in our code.
  • The names of variables, called identifiers, conform to certain rules.
  • Identifiers usually starts with letter, underscore or dollar sign ($)
  • We can also use unicode letters in identifiers.
  • Control flow is the order in which a computer executes code in a script.
  • Scope is the current context of execution in which values and expressions are visible or can be referenced.
  • JS has the following kinds of scope.
    • Global Scope
    • Module Scope
    • Function Scope
    • Block Scope
  • Array is a single variable that is used to hold a group of data.
  • There are two requirements for a JS function:
    • Define the function
    • Call the function
